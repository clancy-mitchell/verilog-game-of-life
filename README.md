# Verilog - Conway's Game of Life

An implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) in Verilog, simulated in real-time with [PyVerilator](https://github.com/csail-csg/pyverilator), and visualized with [Matplotlib](https://matplotlib.org/).
